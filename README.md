# Planet9

Planet9 is a retro-style space shooter game built using [MonoGame](https://www.monogame.net/) and .NET 9. Defend the galaxy against waves of alien invaders in this classic arcade experience.

## Features

- **Classic Arcade Gameplay**: Fast-paced shooting action.
- **Level Progression**: Multiple levels with increasing difficulty.
- **Audio Experience**: Immersive background music for titles and levels.
- **Custom Assets**: Unique ship and alien sprites.
- **Fullscreen Mode**: Toggle fullscreen for an immersive experience.

## Prerequisites

- [.NET 9.0 SDK](https://dotnet.microsoft.com/download/dotnet/9.0)

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/wkdnev/planet9.git
   ```
2. Navigate to the project directory:
   ```bash
   cd planet9
   ```
3. Run the game:
   ```bash
   dotnet run
   ```

## Controls

| Action | Key(s) |
|--------|--------|
| **Move** | `W`, `A`, `S`, `D` or Arrow Keys |
| **Shoot** | `Spacebar` |
| **Toggle Fullscreen** | `Ctrl` + `F` |
| **Exit** | `Esc` |

- C#
